Visual Studio is a software that allows you to open folders of code and edit them. Go to code.visualstudio.com/download to download. options are available for Windows, Mac, and more.


To access the console on Chrome, you do CTRL (CMD for Macbook users) + Shift + I. This will take you to sources. Then, click on console.


To log something on the console, write 'console.log(thingYouWantToLog);' in any JavaScript file. Log text using quotation marks, and log variables by writing their names without anything else, as seen in the example.


Console is useful because it can help you debug. For example, you can log something at the start of a function, then check the console to make sure that function is being properly executed. Check out the console now!
